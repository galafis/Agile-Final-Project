Agile Final Project: E-commerce Product Catalog Backend

This is a demonstration page for the Agile Final Project by Gabriel Demetrios Lafis. This project showcases the implementation of Agile methodologies (Scrum and Kanban) in developing a backend for an e-commerce product catalog using Flask.

Project Overview

The project focuses on the initial Agile planning phases, user story creation, backlog refinement, sprint planning, and execution. It includes a simple Flask API for product management with unit tests.

Technologies Used

  • Python
  • Flask
  • GitHub Issues & Projects
  • Mermaid for diagrams

API Endpoints (Example)

The Flask application provides the following basic endpoints:

  • GET /: Returns a welcome message.
  • GET /status: Returns the application status.
